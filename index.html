About Me

Hi! I am Adebayo Sikiru Olayiwola, a frontend web developer in Lagos, Nigeria. I love Creating Beautiful Websites And Web Applications, My aim is to expand my knowledge in this field and build more beautiful mobile-friendly websites and web applications. After my Highschool graduation from Baptist Boy's High School, I began learning alot about the frontend field in web development and build some couple of project and hosted an online class to create more awareness of the technology industry. I’m a hard working and dedicated professional front-end developer/user interface designer with a keen eye for detail, and a determination to deliver the very highest quality. I take great pride in my work, and I always try to better myself with every project I work on. I like to write semantic, cross browser compatible code by hand that is re-useable, maintainable and easy to understand. I am motivated, enthusiastic and well organised, with a keen eye for beautiful design, primarily based around user interfaces.

Work

Awesome Design Studio

Frontend Dev& UX Designer • ....

Ensuring user experience determines design choices. Developing features to enhance the user experience. Striking a balance between functional and aesthetic design. Ensuring web design is optimized for smartphones. Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly. The challenge associated with front end development is that the tools and techniques used to create the front end of a website change constantly and so the developer needs to constantly be aware of how the field is developing. The objective of designing a site is to ensure that when the users open up the site they see the information in a format that is easy to read and relevant. This is further complicated by the fact that users now use a large variety of devices with varying screen sizes and resolutions thus forcing the designer to take into consideration these aspects when designing the site.
